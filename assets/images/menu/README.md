# Folder untuk Menu Images

Letakkan foto-foto menu Anda di folder ini dengan nama-nama berikut:

## Nama File yang Diharapkan:

1. **nasi_goreng.png** - Untuk Nasi Goreng
2. **mie_ayam.png** - Untuk Mie Ayam
3. **sate_ayam.png** - Untuk Sate Ayam
4. **bakso.png** - Untuk Bakso
5. **soto_ayam.png** - Untuk Soto Ayam
6. **gado_gado.png** - Untuk Gado-Gado

## Format yang Direkomendasikan:

- **Format**: PNG atau JPG
- **Ukuran**: Minimum 200x200px, ideal 300x300px
- **Aspect Ratio**: Square (1:1) untuk hasil terbaik

## Cara Upload:

1. Ambil foto menu Anda
2. Ubah ukuran sesuai rekomendasi (opsional tapi disarankan)
3. Ganti nama sesuai daftar di atas
4. Letakkan file ke folder ini (`assets/images/menu/`)
5. File akan otomatis terbaca di aplikasi

Contoh: `assets/images/menu/nasi_goreng.png`
